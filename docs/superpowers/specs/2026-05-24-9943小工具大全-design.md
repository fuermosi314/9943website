# 9943小工具大全 - 设计文档

## 项目概述

**项目名称：** 9943小工具大全

**项目目标：** 创建一个综合效率工具网站，提供图片压缩、PDF处理、文本翻译、二维码生成等各类在线工具，方便自己和朋友日常使用。

**技术栈：** Next.js 14 (App Router) + TailwindCSS + TypeScript

**设计风格：** 参考 4399 小游戏网站，采用紧凑卡片网格布局，橙色主题色。

---

## 设计原则

1. **简洁优先：** 首页只保留核心元素（Logo、搜索框、分类导航、工具卡片）
2. **易于扩展：** 每个工具是独立页面文件夹，添加新工具只需注册元数据 + 创建页面
3. **本地优先：** 收藏、历史等功能使用 localStorage，无需登录系统
4. **性能优先：** 纯前端处理，无后端依赖，Vercel 一键部署

### 组件说明

**1. 顶部导航栏**
- Logo：左侧，网站名称 "9943小工具大全"
- 搜索框：右侧，支持模糊搜索工具名称/描述

**2. 分类导航栏**
- 横向排列的分类标签
- 分类：全部、图片工具、文档工具、文本工具、开发工具、生活工具
- 点击切换分类，高亮当前选中

**3. 工具卡片网格**
- 卡片尺寸：120×100px
- 图标：48×48px，居中
- 名称：14px，加粗
- 一行 5-6 个卡片
- 悬停效果：卡片上浮 + 边框高亮

| 文本工具 | 📝 | 文字转语音、字数统计、文本对比、JSON格式化 |
| 开发工具 | 🔧 | Base64编解码、URL编解码、正则测试、颜色转换 |
| 生活工具 | 🎯 | 二维码生成、随机数生成、计算器、时间戳转换 |
