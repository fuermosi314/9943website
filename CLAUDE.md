# 9943小工具大全

## 强制规则：先读后做，做完必写
1. **开始前** — 先读 [PROJECT.md](PROJECT.md)，理解完整设计规范再动手
2. **结束后** — 任何改动（新增工具、修改分类、调整样式、修复 bug）必须同步更新 PROJECT.md

## 快速命令
- `npm run dev` — 启动开发服务器 (http://localhost:3000)
- `npm run build` — 生产构建

## 核心规则
- 主色 `#fb6400` 不可更改，"9943小工具大全"标题保持橙色
- 工具页面返回必须回到上级分类，不能跳回"全部工具"
- 新增工具必须在 `lib/tools.ts` 注册，网站工具还需在 `site/[slug]/page.tsx` 添加 features
- 所有工具卡片框尺寸必须一致
- 工具页面需包含统一的导航栏和背景效果
